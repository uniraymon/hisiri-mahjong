怎麼接到 Claude Code 的遊戲

每個 SVG 都是 200×250 viewBox，全部對齊好；你只要把它們疊在同一個容器裡（絕對定位），盔甲光暈放最底層，角色底圖中間，武器/飾品最上層。範例：

<div class="paper-doll">
  <img class="layer armor-glow" src="assets/equipment/armor-glow.svg"/>
  <img class="layer base"       src="assets/classes/warrior.svg"/>
  <img class="layer weapon"     src="assets/equipment/sword.svg"/>
  <img class="layer accessory"  src="assets/equipment/crown.svg"/>
</div>

<style>
.paper-doll { position: relative; width: 200px; height: 250px; }
.paper-doll .layer { position: absolute; inset: 0; width: 100%; height: 100%; }
</style>

或在 React 裡：

function PaperDoll({ klass, weapon, accessory, armor }) {
  return (
    <div className="relative w-[200px] h-[250px]">
      {armor && <img src={`/assets/equipment/${armor}.svg`} className="absolute inset-0" />}
      <img src={`/assets/classes/${klass}.svg`} className="absolute inset-0" />
      {weapon && <img src={`/assets/equipment/${weapon}.svg`} className="absolute inset-0" />}
      {accessory && <img src={`/assets/equipment/${accessory}.svg`} className="absolute inset-0" />}
    </div>
  );
}
